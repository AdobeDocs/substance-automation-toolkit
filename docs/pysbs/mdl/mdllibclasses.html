mdllibclasses moduleÔÉÅ

Module mdllibclasses provides all the classes used by the MDL library for the definition of all MDL entities. In particular, it defines the classes:

  • MDLNodeDef

  • MDLNodeDefInput

  • MDLNodeDefOutput

  • MDLNodeDefParam

  • MDLNodeDefParamValue
