modeloperand module

Module moduleoperand aims to define SBSObjects that are relative to a model operand, Each class handle an operand type

class modelgraph.modeloperand.ModelBaseOperand(**kwargs)

Bases: ModuleOperand

ModelBaseOperand is the base class for operand, each typed operand inherited of ModelBaseOperand

Members:

  • mName (str): operand’s name.

  • mType (str): operand’s type.

  • mValue (str): operand’s value.

  • mDefaultValue (str): operand’s value.

  • mOperandMetaData (ModelOperandMetaData): operand’s metadata object.
